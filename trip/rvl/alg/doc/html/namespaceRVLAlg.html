Detailed Description

boolterm.H All Terminators which manipulate other terminators using boolean logic.

vectorterm.H Terminators which involve manipulating Vectors.

scalarterm.H All Terminators which watch the behavior of a scalar or integer to determine when to stop.

ioterm.H All Terminators which perform some I/O when queried.

This includes terminators which prompt the user for some action as well as those which print out messages and tables.

Includes using the built-in Vector operations as well as passing vectors to other objects.
