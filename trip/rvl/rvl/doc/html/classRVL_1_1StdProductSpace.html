Detailed Description

template<class Scalar>
class RVL::StdProductSpace< Scalar >

Standard construction of product spaces, via an (STL) vector of Spaces.

Note that all data references are to external objects. Therefore copy is shallow. No data other than pointers is owned.

In order avoid dangling pointer issues, the only usable constructors take explicit lists of const space references. Thus StdProductSpace can be instantiated only within the scope of the component Space instantiations.

Convenience constructors supplied for 2, 3, and 4 components. Arbitrary products may be built up via the associativity of Cartesian product.
