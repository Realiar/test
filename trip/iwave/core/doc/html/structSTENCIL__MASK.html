Detailed Description

Mask structure.

stencil

Stencil: finite difference stencil description object.

Stencil is described by masks. Each mask [ip,ir] describes which grid points of array ip are used for computation of array ir. It is not required to store empty masks.

Requirements:

  • Along each coordinate axis all arrays have same grid step. Grid step can be different for different axes.

  • Arrays can be shifted relative to each other (e.g. staggered). Shifts are not reflected in the stencil info and array indices are always integer.

  • It is user's responsibility to make assumptions about shifts.

Example: Acoustic wave equations:

2-4 staggered grid, 2D.

  • Assumptions:
    • Pressure with index (0,0) corresponds to ( 0 , 0 ) point.
    • Velocity with index (0,0) corresponds to (1/2, 0 ) point.
    • Velocity with index (0,0) corresponds to ( 0 ,1/2) point. 
        E.g. (see first mask below):
      

     variable:   P   P V P   P 
        index:  -1   0 0 1   2 
      stencil:  -o---o-X-o---o-
